# Ethernet Cabling & Color Coding

## Objective
Understand Ethernet cable types (straight-through vs crossover) and the 
TIA/EIA-568 wiring color-code standards used to terminate them.

## Note on this folder
This topic was covered as an in-class/hands-on cabling exercise and was not 
submitted as a digital Packet Tracer file. It is documented here for 
completeness, since it is a foundational prerequisite for every lab that 
follows in this repository.

## Concepts Covered
- TIA/EIA-568A vs 568B wiring standards
- Straight-through cable use case (dissimilar devices — PC to Switch)
- Crossover cable use case (similar devices — PC to PC, Switch to Switch)
- Auto-MDIX (why modern switches often don't require crossover anymore)
